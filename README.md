# 作业跟踪

一个面向班主任日常使用的作业缺交记录小工具，包含：

- Flask 数据看板（可视化趋势、学科占比、学生排行榜）
- Web 端快速录入（智能解析文本，自动识别学科）
- 数据管理页面（编辑、删除记录）
- 学生个人详情页（点击排行榜跳转）
- 学期配置（自定义统计区间）
- SQLite 本地数据存储
- Excel 学生名单导入 & 每日缺交记录自动导出

项目默认用于本地管理班级作业缺交情况，不依赖云端服务。

## 功能说明

### 1. 数据看板

启动 Web 服务后，可以查看：

- 缺交总人次
- 重灾区学科
- 需重点关注学生（Top 5）
- 每日缺交趋势图（点击数据点可筛选单日）
- 各科缺交占比饼图（悬浮显示学生明细）
- 学生缺交次数排行榜（点击跳转学生个人页面）
- **多维度筛选**：支持按日期区间、学科名称、学生姓名进行灵活的数据过滤

其中学科统计已做归类处理，例如：

- `英语粉书`、`英语卷子`、`英语笔记` 会统一计入 `英语`
- 其他学科也会按学科关键词归并

### 2. Web 端快速录入

在看板页面顶部可以直接录入缺交记录，支持批量文本格式：

```text
张三: 英语粉书
李四: 数学、物理笔记
王五: 请假
```

系统会自动解析：
- 学科识别：`英语粉书` → 学科 `英语`，说明 `英语粉书`
- 特殊情况：`请假` → 学科 `全科`，特殊情况 `请假`

### 3. 数据管理

通过 `/manage` 页面可以：

- 按日期、学生筛选记录
- 编辑单条记录的学科、说明、特殊情况
- 删除记录

修改后会自动重新导出对应日期的 Excel 报表。

### 4. 学生个人页面

从排行榜点击学生名字可跳转到 `/student/姓名` 页面，查看该学生在当前筛选时间段内的缺交明细。

### 5. 学期配置

通过 `/settings` 页面可以设置学期的起止日期，看板会默认使用学期区间作为筛选范围。

### 6. 学生名单导入

支持从 Excel 导入学生名单，自动识别常见列名：

- `学号`
- `姓名`
- `性别`

### 7. 每日缺交表导出

录入记录后，会自动按日期导出 Excel 版缺交记录，目录结构类似：

```text
2026/
  03月/
    2026-03-13缺交记录.xlsx
```

## 环境准备

建议使用虚拟环境。

### 安装依赖

```bash
pip install flask pandas openpyxl
```

## 快速开始

### 1. 初始化数据库

首次使用可执行：

```bash
python tracker.py init
```

### 2. 导入学生名单

```bash
python import_excel.py "2025级6班名单.xlsx"
```

### 3. 启动看板

```bash
python app.py
```

或在 macOS 上双击 `启动看板.command` 文件。

默认访问地址：

```text
http://127.0.0.1:5000
```

## 页面导航

| 页面 | 路径 | 说明 |
|------|------|------|
| 数据看板 | `/` | 主页，含录入、筛选、图表 |
| 数据管理 | `/manage` | 编辑、删除记录 |
| 学生详情 | `/student/姓名` | 学生个人缺交明细 |
| 学期配置 | `/settings` | 设置学期起止日期 |

## Web API

看板与管理页面使用以下接口：

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/kpi` | GET | KPI 统计 |
| `/api/trend` | GET | 每日缺交趋势 |
| `/api/subjects` | GET | 学科占比及学生明细 |
| `/api/rankings` | GET | 学生排行榜 |
| `/api/records` | POST | 批量录入记录 |
| `/api/manage/records` | GET | 查询记录列表 |
| `/api/manage/records/<id>` | PUT | 更新记录 |
| `/api/manage/records/<id>` | DELETE | 删除记录 |
| `/api/semester` | GET/PUT | 学期配置 |

看板接口支持的查询参数：`start_date`、`end_date`、`student`

## 常用命令

### 录入一条缺交记录

```bash
python tracker.py add_record --name "张三" --date 2026-03-13 --subject "英语粉书"
```

### 查询某个学生的缺交记录

```bash
python tracker.py query --name "张三"
```

### 按日期范围查询

```bash
python tracker.py query --name "张三" --start 2026-03-01 --end 2026-03-31
```

### 查看全部学生

```bash
python tracker.py list_students
```

### 手动导出某天的缺交表

```bash
python tracker.py export --date 2026-03-13
```

## 数据与隐私说明

仓库已通过 `.gitignore` 默认忽略以下内容，不会自动提交到 GitHub：

- `homework.db`
- 学生名单 Excel
- 每日导出的 Excel 记录
- `venv`

这部分数据通常包含学生个人信息，建议只保留在本地。
